## Scope

Story 19.0's deliverable is the FRAMEWORK-PROVIDED read tool `iris_server_profiles`,
registered centrally in `@iris-mcp/shared` `server-base.ts` (decision E1). It reports the
profile roster (password NEVER serialized) and the effective governance policy (via the
shared `getEffectivePolicy`, the same source the D6 resource uses). There is NO
ObjectScript / live-IRIS surface — the tool reports in-memory config and does not connect.
All tests run against a constructed `McpServerBase` with synthetic env, no live IRIS.
