**(2) `server` value edge cases (empty / undefined / whitespace):**

- [x] Whitespace-only `server` (`"   "`) is NOT treated as default → structured unknown-profile `isError` (name echoed verbatim, valid names listed, no establishment fetch issued). Confirms `resolveProfile` special-cases only `""`/`undefined`.
